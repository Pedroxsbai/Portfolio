Smart and Secure Pharmaceutical Management Solution

Project Overview

This project focuses on the development of an intelligent pharmaceutical stock management system that combines advanced analysis, strong software design, and secure implementation. It was built to support pharmacy-related operations with a structured and collaborative approach, while addressing both functional and non-functional requirements.

The system was modeled using UML for business process visualization and Merise for data structuring and database design. The project includes actor and entity modeling, client-server architecture design, DBMS selection, relational schema generation, and SQL script design.

On the implementation side, the solution uses a dynamic web module built with PHP, AJAX, and prepared statements to ensure security, responsiveness, and effective handling of sensitive data. It also includes pharmaceutical and stock management features, along with an AI chatbot specialized in answering pharmaceutical-related questions.

The project reflects a strong balance between modeling, architecture, software quality, and performance, while remaining extensible for future features such as real-time notifications and audit logs.
